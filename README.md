# 算法实现

![](https://img.shields.io/badge/java-SpringBoot-orange)
该项目基于Spring Boot结合thymeleaf模板引擎实现，实现了6个常见的数值分析算法。
## 使用
```java
git clone https://github.com/Venom-lemon/Numerical-Analysis.git
```
直接在本地启动即可运行
